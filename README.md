# Aritya-arjunan Crawl

## How do I install these formulae?

`brew install aritya-arjunan/crawl/<formula>`

Or `brew tap aritya-arjunan/crawl` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aritya-arjunan/crawl"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
